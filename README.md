# TwentyOne Console Application

## Author
Manmeet Singh Mangat

## Description
This is a C# console application based on the game Twenty-One (Blackjack).  
The purpose of this project is to demonstrate debugging, object-oriented programming concepts, and understanding program flow using Visual Studio.

## Features
- Console-based gameplay
- Player and Dealer classes
- Card and game rule logic
- Use of lambda expressions and LINQ
- Step-by-step debugging using breakpoints

## Technologies Used
- C#
- .NET 6.0
- Visual Studio

## How to Run
1. Open the `TwentyOne.sln` file in Visual Studio.
2. Press **F5** to run the application.
3. Use **F11 (Step Into)** to debug the program line-by-line.

## Assignment Requirements Covered
- Debugging with breakpoints
- Understanding program execution flow
- Explaining specific lines and blocks of code
- Object-oriented programming principles

## Notes
This project was created for educational purposes and fulfills the requirements of the debugging and code analysis assignment.
